template<typename T>
class TManager< T >

Implémente la classe manager.

Author
DELAUNAY Grégory
Version
1.0
Date
04 juin 2015

Cette classe définit les manager génériques Les manager sont des objets singleton qui servent à gerer le cycle et l'utilisation des types associés.
