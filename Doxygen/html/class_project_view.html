Detailed Description

Implémente la classe ProjectView.

Author
DELAUNAY Grégory et NEVEUX Anaïs
Version
1.0
Date
08 juin 2015

Ce widget est l'un des widgets principaux de l'application. Il sert à gérer tout ce qui est création et édition de projet, et de tâches. Il permet également de gérer les liens entre les différentes entités.
