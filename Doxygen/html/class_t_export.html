Detailed Description

template<typename U>
class TExport< U >

Implémente le template TExport.

Author
NEVEUX Anais
Version
1.0
Date
12 juin 2015

Le template TExport agit comme une interface devant etre implémentée pour tout export, elle permet également de mettre en place la stratégie d'exportation, d'une part car elle généralise ce qu'on veut exporter, mais également car elle stocke le format d'exportation.
