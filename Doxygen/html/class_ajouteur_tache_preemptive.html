Detailed Description

Implémente la classe AjouteurTachePreemptive.

Author
DELAUNAY Grégory
Version
1.0
Date
04 juin 2015

Cette classe définit les ajouteurs de programmation de tâche unitaire preemptive Les ajouteurs sont des objets qui servent à construire des objets d'un type données. Ils sont utiles pour que les managers puissent gérer de nouveaux types d'objets sans avoir à tout recompiler.
